# agile4j-kts-loader

## 如何引入

>Gradle
```groovy
dependencies {
    compile "com.agile4j:agile4j-kts-loader:1.0.0"
}
```
>Maven
```xml
<dependency>
    <groupId>com.agile4j</groupId>
    <artifactId>agile4j-kts-loader</artifactId>
    <version>1.0.0</version>
</dependency>
```

## 使用方式
```Kotlin
eval("src/test/resources/test-script.kts")
```
